# Week 1 - LLM Benchmark

This repository contains my Week 1 AI Internship assignment.

## Objective

The objective of this project is to compare different open-source Large Language Models (LLMs) by measuring their inference time, GPU memory usage, and response quality using Google Colab.

## Models Tested

- Qwen2.5-0.5B
- Zephyr-7B-beta
- Phi-3.5-mini-instruct
- DeepSeek-R1-Distill-Qwen-1.5B
- TinyLlama-1.1B-Chat

## Tools Used

- Python
- PyTorch
- Hugging Face Transformers
- Google Colab

## Results

The benchmark results are included in the notebook and exported as CSV files.
